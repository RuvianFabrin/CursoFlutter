Instalando o Flutter
Vá para https://flutter.dev/
Clique em Get Start
você pode baixar o zip ou fazer um clone com o Git
Depois que baixou os arquivos, inclua o flutter nas variaveis de ambiente, no path
Insira: C:\flutter\bin
Verifique se a pasta existe

no Power shell digite
flutter e aguarde
depois flutter doctor

Instale o android studio

digite flutter doctor no power shell

Verifique se tem mensagem com a palavra Run
se sim, pesquise como resolver

Criando o Primeiro Projeto
Você precisa de um emulador ou celular para rodar o projeto

flutter emulators , vai mostrar os emuladores disponiveis

flutter emulators --lauch nome_do_emulador

flutter create nome_projeto // para criar um projeto

abra a pasta pelo visual studio code

lib/main.dart //é o arquivo principal do projeto

se já está configurado o celular ou emulador, abra o terminal no visual estudio e digite flutter run
